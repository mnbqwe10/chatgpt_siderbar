# AI Sidebar

A Chrome extension that opens popular AI chatbots in a side panel and lets you send selected text from any page with one click.

## Install

1. Get the project files:
   git clone https://github.com/mnbqwe10/chatgpt_siderbar.git
   or download the repository ZIP from GitHub and unzip it
2. Open chrome://extensions/
3. Turn on **Developer mode**
4. Click **Load unpacked**
5. Select the chatgpt_siderbar folder

## Supported Platforms

- ChatGPT (OpenAI)
- Microsoft 365 Copilot
- Google Gemini
- DeepSeek
- Claude (Anthropic)

## Get Started

1. Open the extension and sign in to your preferred platform in the sidebar
2. Select text on any page to open the quick toolbar
3. Use **Open**, **Copy**, **Explain**, **Translate**, **Summarize**, or **Ask**
4. Switch platform directly from the dropdown in the sidebar header

## Options

Open the extension's **Options** page to:

- choose the default AI platform
- change the built-in Explain, Translate, and Summarize prompts
- choose the Translate target language
- block sites where the toolbar should not appear

Use the extension popup from the Chrome toolbar to:

- enable or disable the selection toolbar
- choose the active AI platform
- choose the Translate target language
- customize which toolbar buttons appear, their order, and whether labels are shown
- open the AI sidebar

Prompt templates support:

- {{selected_text}}
- {{page_url}}
- {{page_title}}
- {{target_language}}
